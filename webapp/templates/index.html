Turn any CSV into clean insights.

Upload any CSV file — the dashboard auto-detects your column types and generates as many charts as your data supports. No specific schema required.

  • • Auto-detects dates, numbers, and categories — no config needed
  • • Generates histograms, time-series trends, bar/pie charts, scatter & correlation heatmaps
  • • Dynamic filters built from whatever categorical columns exist

Upload a CSV

✅ Minimum (always works)
  • At least one column of any type
  • A valid, non-empty CSV file (UTF-8 or common encodings)
  • Header row in the first line

Even with just one column you'll get: a numeric summary table, row count KPI, and a histogram (if the column is numeric).

⚡ Better with numeric columns

Any column that looks like a number (or is named like one below) unlocks:

  • Histograms for each numeric column
  • Scatter plot (2+ numeric columns)
  • Correlation heatmap (3+ numeric columns)
  • Sum KPI cards

Auto-recognised names (any variant):

Sales / Revenue / Amount / Profit / Margin / Quantity / Discount / Price / Value
🚀 Even better with dates & categories

Date column (name contains date / time / created / timestamp):

  • Monthly time-series trend charts
  • Year / Month filter dropdowns
  • Month-over-month % change metrics

Categorical columns (≤ 50 unique values):

  • Bar & pie charts per category
  • Filter dropdowns for each category

Auto-recognised names: Region / Category / Segment / Customer / Product and their common variants.
